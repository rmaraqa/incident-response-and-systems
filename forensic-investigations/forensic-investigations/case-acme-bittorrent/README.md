Case Overview

This case involved a corporate digital forensic investigation into alleged copyright infringement within an enterprise environment.
The investigation focused on determining whether an employee violated company policy by using peer-to-peer file sharing software within a virtualized system.

 Investigative Focus

- Analysis of virtual machine environments
- BitTorrent client and log analysis
- Correlation of file system artifacts, application logs, and browser history
- Differentiation between legitimate and policy-violating usage

Approach

The investigation followed a structured methodology:
- Validation of forensic images and integrity checks
- Examination of BitTorrent client artifacts and configuration files
- Correlation of torrent metadata with downloaded content via hash analysis
- Timeline reconstruction of user behavior across applications and systems

Key Findings

- Multiple BitTorrent clients were installed and actively used
- Evidence of torrent activity extending beyond legitimate software downloads
- Artifact consistency supporting intentional use of peer-to-peer networks
- Attempts to obscure activity through torrent removal and configuration settings

Skills Demonstrated

- Enterprise forensic analysis
- Log and application artifact correlation
- Policy-focused investigative reasoning
- Clear attribution under competing explanations
